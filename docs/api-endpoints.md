# API Endpoints
------

## HTML API
------

### Root
- `GET /`

##JSON API
------

### Users
- `POST /api/users`
- `PATCH /api/users`

### Sessions
- `GET /api/session`
- `PATCH /api/session`
- `DELETE /api/session`

### Profiles
- `GET /api/profiles/:id`
- `POST /api/users/:id/profiles`
- `PATCH /api/profiles/:id`

### Events
- `GET /api/events`
- `GET /api/events/:id`
- `POST /api/events`
- `PATCH /api/events/:id`
- `DELETE /api/events/:id`
